# TCP/UDP Sockets

## Overview
Network sockets allow communication between computers over a network. They provide an interface for sending and receiving data.

## TCP (Transmission Control Protocol)
- Connection-oriented
- Reliable, ensures data delivery
- Data is received in order
- Example: Web browsing, email

## UDP (User Datagram Protocol)
- Connectionless
- Faster but less reliable
- Data may arrive out of order or be lost
- Example: Video streaming, online gaming

## Key Differences
| Feature | TCP | UDP |
|---------|-----|-----|
| Connection | Yes | No |
| Reliability | High | Low |
| Order of Data | Maintained | Not guaranteed |
| Speed | Slower | Faster |
